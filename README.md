```
3 나오는게 정상임

int a = 2; --> int(숫자) 형 변수 a 선언 하여 2을 집어 넣는다.
a= 3       --> 이미 선언한 a에 3를 집어 넣는다.

int a = 3;  --> int(숫자)형 변수 a를 또 선언해서 3을 넣는다. (이미 선언한 a는 다른 디스크에 떠다니고있다가 버려짐) 


sysout(a)   --> 3이 나온다.

```
